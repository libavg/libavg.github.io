Planarity¶

Planarity is a multitouch adaption of the popular game Planarity, also known as Untangle. Each level starts with a tangled net of connected vertices. To solve a level, the player has to move the vertices around until no edge clashes are left, i.e. the net is untangled.

Written by Martin Heistermann, additional code by Thomas Schott & Ka-Ping Yee.

Planarity is Open Source and licensed under the GPL. You can get the source code using

git clone https://github.com/libavg/mtc-planarity.git

There is an Ubuntu package for Planarity.
